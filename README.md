# BUAA_CO_2023

该仓库代码已通过课上所有测试，供学习参考，以使你能设计出更好的CPU。

笔者在考试完后尝试竞选了计组助教，但不幸没能选上，没法刁难xdx们了(，还是挺遗憾的。
